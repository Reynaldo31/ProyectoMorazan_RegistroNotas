# ProyectoMorazan_RegistroNotas
Aplicación de consola para el registro de notas de estudiantes

# Desarrollador
 * Reynaldo Jafet Giron Tercero

# Abrir el Proyecto en Visual Studio
 * Abrir Microsoft Visual Studio.
 * Seleccionar "Abrir un proyecto o una solución".
 * Navegar al directorio donde se clono el repositorio y abrir el archivo .sln correspondiente del proyecto.
  
# Ejecutar la Aplicación
 * Seleccionar el proyecto de consola en el Explorador de Soluciones (Por lo general este se encuentra en la lista de proyectos a la 
   izquierda).
 * Dar clic en el botón "ProyectoMorazan_RegistroNotas" (Es un boton verde como un triángulo) o presiona F5 para ejecutar la aplicación en 
   modo depuración.
 * Si se desea ejecutar la aplicación sin depuración, se puede presionar Ctrl + F5.
